Есть JSON с 10к транзакциями десяти пользователей. Нужно реализовать приложение на golang, которые считает JSON и
смормирует отчёт.

Level 1: полученный отчёт отобразить в консоли.   Минимальный функционал чтобы засчитать задание.

Level 2: полученный отчёт сохранить в JSON файле.

Level 3: полученный отчёт сохранить в CSV файле.

Level 404: можно пропустить сохранение в файле, а сохранить данные отчета в Database и через SQL запрос в консоли.
Читаем JSON -> формируем структуру данных отчёта -> сохраняем в бд


Основным критерием оценки станет время выполнения программы.  Code style и как оформлен репозиторий: readme, makefile,
docker, tests, benchmark.
Результат работы публикуем в GitHub и присылаем ссылки на репозиторий.
